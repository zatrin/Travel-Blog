# Travel-Blog
1.Travel Blog

Welcome to my Travel Blog website! This project is a simple and responsive travel-themed website built using HTML and CSS.

The website is designed to showcase beautiful travel destinations with a clean layout and an enjoyable user experience. It helped me practice web design, page structure, and responsive styling.

2. Features
Clean and modern design
Responsive layout
Travel destination sections
Easy navigation
Attractive images and content presentation

3.Technologies Used

HTML5
CSS3
 4.Purpose of the Project

I built this project to improve my front-end development skills and gain hands-on experience with HTML and CSS. It allowed me to practice creating responsive layouts and designing visually appealing web pages.
